HOW TO USE
================================================================================

Normal User Flow:
   1. Open http://localhost:3000
   2. Click "Node A" or "Node B" tab
   3. Click "Upload" tab
   4. Enter Object ID (e.g., "my-cube")
   5. Drag-drop or select a 3D file
   6. Click "Upload" button
   → System creates version 1
   7. Upload another version
   → System creates version 2
   8. Click "History" tab
   9. Compare versions
   → View detailed geometry diff

Developer Flow:
   1. Check API documentation in FRONTEND_GUIDE.md
   2. Use REST endpoints directly with curl/Postman
   3. Customize backend logic in node_a/src/, node_b/src/
   4. Modify React components in frontend/src/
   5. Deploy with docker-compose or manual setup
